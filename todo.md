## 阶段 6:手感打磨(当前重点)

> 顺序建议:6A 流程细节 → 6B 换图 → 6C 音乐音效。每块做完都可独立试玩验证。

### 6A. 流程细节打磨(动画 + 反馈 + 边界修复)
**交互手感**
- [ ] 拾取动画:器械平滑抬起到光标(Tween,而非瞬移)
- [ ] 放置正确:器械落入槽位的"落定"缓动 + 轻微缩放回弹
- [ ] 放置错误:红闪 + 器械弹回手中(Tween 弹簧感)
- [ ] 槽位 hover 高亮:光标对准槽位时边框高亮(可放置提示)
- [ ] 器械 hover:光标对准器械时轻微放大/描边(可拾取提示)

**医生手动画**
- [ ] 伸/收手加缓动(目前线性 Tween,改 ease)
- [ ] 接住器械时"抓握"动效(手 mesh 缩放或夹合)
- [ ] 拒绝时"推开"动效(手后缩再回)
- [ ] 归还时手伸出 + 器械可见在手中

**流程边界与 UX**
- [ ] 最后一件器械放回后才结算(确认当前逻辑:place_back 触发 finish_surgery)
- [ ] 递送文字时机复查:仅医生再次伸手要下一件时显示
- [ ] 取回提示"取回:X"在归还阶段清晰可见
- [ ] 倒计时"开始!"后留短暂缓冲再出第一个需求
- [ ] 结算页:星级依次弹出动画 + 用时/正确率逐行显示
- [ ] 重新开始按钮确认能正确重置所有状态(器械/槽位/分数/阶段)
- [ ] 准备阶段:未全部归位时给一个"还差 N 件"的进度提示
- [ ] 防卡死:持物时若点击空处,器械不丢失;极端情况兜底

**氛围**
- [ ] 托盘随器械被拿走逐渐变空的视觉(满足感)
- [ ] 槽位全部填满时台面整体高光/完成感
- [ ] 步骤完成时一个小进度条/计数推进动效
